== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Description ==
Remax Store is modern, clean, colorful and responsive store theme. It can be used for blogs, informative, news and other creative websites. It uses an amazing WordPress Customizer for theme options.Demo: http://demo.themerelic.com/remax-store/


== Frequently Asked Questions ==

remax-store includes support for Infinite Scroll in Jetpack.
